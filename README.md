# Gemini-Search-MD-Copier

Google検索のAIによる回答やGeminiの出力結果を、ワンクリックで綺麗なMarkdown形式にしてクリップボードにコピーするChrome拡張機能です。

## 概要 (Overview)

Google検索上部のAI要約やGeminiの回答は非常に便利ですが、ドキュメントアプリ（Notion, Obsidian, GitHubなど）に貼り付ける際、書式が崩れてしまうことがよくあります。
この拡張機能を使用すると、AIの回答を自動的にMarkdown構文に変換し、ブラウザ右上のアイコンをクリックするだけで瞬時にクリップボードに保存できます。

## 対応サービス (Supported Services)

以下のAIアシスタントの出力に対応しています：
* Google検索のAIによる要約（通常検索の上部表示）
* Google検索のAIモード
* Google Gemini

## 主な機能 (Features)

* **ワンクリックコピー**: ブラウザの拡張機能アイコンをクリックするだけのシンプル操作。
* **Markdown書式の保持**: 見出し、リスト、太字、コードブロックなどのテキスト構造を維持してMarkdown化。
* **軽量・高速**: Manifest V3とVanilla JavaScriptのみで構築されたシンプルな設計。

## インストール方法 (Installation)

1. このリポジトリをクローン、またはZIP形式でダウンロードして解凍します。
2. Chromeブラウザを開き、アドレスバーに `chrome://extensions/` と入力して拡張機能管理ページにアクセスします。
3. 画面右上の「デベロッパーモード」をオン（有効）にします。
4. 左上の「パッケージ化されていない拡張機能を読み込む」をクリックし、ダウンロードしたリポジトリのフォルダを選択します。
5. ブラウザ右上の拡張機能メニュー（パズルアイコン）から、本拡張機能をピン留めしておくと便利です。

## 使い方 (Usage)

1. 対応しているページ（Google検索のAI回答エリア、またはGemini）を開き、AIの回答を表示させます。
2. ブラウザ右上の拡張機能アイコンをクリックします。
3. 「マークダウン形式でクリップボードにコピーしました！」のダイアログが表示されるので、OKをクリックしてください。
4. お好きなエディタなどにMarkdownテキストとしてそのまま貼り付けてください。
5. 対象でないページで拡張機能をクリックしても実行しないようになっています。

## 技術スタック (Tech Stack)
